# Homiely_CS35L
Simplify your roommate problems!

## Setup

### Local Development

#### Frontend
```bash
cd frontend
npm install
npm start
```

#### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Docker Deployment
```bash
docker-compose up --build
```

Frontend will be at http://localhost:3000
Backend API at http://localhost:5000
Database at localhost:5432
